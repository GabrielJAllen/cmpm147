Experiment 3 - Alternate Worlds

Description

This experiment is attempting to create a procedurally generated tiled scene that has water with animation and autotiling. The scene is intended to be a portion of a pond or lake in a forest

Technical

So the two important parts of the Technical were the text generation for the water and trees, and the animation for the water. The text generation first adds in the water start at the top of the grid and as it goes down each row it may add to the width of the water on one side. If it hits the edge it will always add onto the none touching edge. Then the trees are added on both sides a random distance away from the water characters. Due to a bug the trees are at an irregular distance but it works well for looking natural so it wasn't fixed. The animation function is my partner's creation, but here it is used to loop through a set of the water tiles to simulate the water moving. It indexs the array of tile coordinates based off the frame count currently.
